# 🎮 Simon Game (JavaScript)

An interactive browser-based Simon memory game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Dynamic sequence generation with increasing difficulty
- Real-time user input validation
- Event-driven gameplay using JavaScript
- Visual feedback for better user interaction

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation, Event Handling)

## 🎯 How it works
The game generates a random sequence of colors that the user must remember and repeat.  
With each level, the sequence increases, making the game more challenging.

## ▶️ Run Locally
1. Clone the repository  
2. Open `index.html` in your browser
